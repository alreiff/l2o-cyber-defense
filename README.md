# l2o-cyber-defense
The goal of this project is to develop scalable optimization and learning methods for cybersecurity defense problems modeled as **bi-level optimization problems**, such as:

- honeypot placement in attack graphs
- shortest-path interdiction
- sensor placement
- resource allocation against strategic attackers

The repository will progressively contain:

- optimization solvers
- learning-to-optimize architectures
- graph neural network models
- benchmark environments
- experimental notebooks
